Installation
============

.. code-block::

   pip install pyharm

Alternatively, you can run the following command with the test option to execute regression tests:
   
.. code-block::
   
   pip install --install-option test

The only dependencies are PyTorch and NumPy.

Explore
============

Check out the `examples <https://github.com/ivanZanardi/pyharm/tree/main/examples>`_ provided in the repository to see PyHarm in action.
